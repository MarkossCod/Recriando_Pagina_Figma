Skills

FrontEnd

Front-end é a parte visual e interativa de um site ou aplicativo — tudo aquilo que o usuário vê e com o que interage. Envolve HTML, CSS e JavaScript para criar layouts, cores, botões, animações e toda a experiência na tela. É basicamente a “interface” que conecta o usuário com o sistema.

BackEnd

Backend é a parte interna de um sistema que processa dados, aplica regras de negócio e se conecta ao banco de dados. Ele garante que tudo funcione por trás das telas, respondendo às ações do usuário e mantendo a aplicação segura e eficiente.
